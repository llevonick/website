Universal Machine Emulator

Created a software implementation of a virtual machine in C that could run any file that contained instructions in binary. The code reads in multiple instructions and runs them to do potentially anything. This code was later optimized, allowing an executable with 2 billion instructions to run in 14 seconds when it originally ran in 2 minutes.

MBTA Map

Image Compression/Decompression

Developed a C program that can compress a pgm file or convert a compressed file into a pgm. Accomplished this by shifting and modifying bits to pack pixel data into 4-byte codewords, and by reversing the process.

Image Processing
